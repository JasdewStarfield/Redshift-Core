# **REDSHIFT (红移) \- Modpack Design Document**

Version: 1.0 (Pre-Alpha Concept)  
Platform: Minecraft 1.21.1 NeoForge  
Genre: Industrial Builder / Survival / Narrative Adventure  
Theme: Atom Punk, Cassette Futurism, Hard Sci-Fi, Physics Horror  

## **1\. 游戏概述 (Overview)**

**《REDSHIFT》** 是一个以剧情和探索为导向的 Minecraft 工业整合包。玩家在一个与世隔绝的异星系冷战废墟中苏醒，通过“工业考古”手段修复生产线，在美苏双方遗留的自动化战争机器（死手系统）夹缝中求生。

核心体验在于：**依靠发掘出的科技，从勉强求生开始，一步一步发掘这个星系的真相。玩家会看到战争和绝望阴霾下暴露无遗的卑劣，也能看到其中闪耀的人性。**

### **核心支柱 (Core Pillars)**

1. **工业考古 (Industrial Archaeology)**：科技不是凭空研发出来的，而是从废墟中挖掘、修复和逆向工程得来的。  
2. **双极工艺 (Bipolar Crafting)**：玩家需要根据环境，在苏式（耐用、抗干扰、笨重）和美式（高效、精密、脆弱）两种技术路线中权衡和混搭。  
3. **物理恐怖 (Physics Horror)**：恐惧不来自怪兽，而来自异星复杂危险的自然环境以及“红素粒子”（负质量/引力坍缩）的不可控。  
4. **磁带未来主义 (Cassette Futurism)**：所有UI和交互都必须呈现出80年代末的物理质感（CRT 屏幕、机械开关、磁带噪声）。

## **2\. 叙事概要 (Narrative)**

### **背景设定**

* **时间**：1990年后的平行世界。  
* **事件**：冷战后期，由于虫洞的开启，美苏在异星系发现了高能物质“红素粒子”，也引发了双方的争夺。  
* **大崩溃**：随着竞争白热化，为了防止武器化的西格玛粒子引发地球重力井的“真空衰变”，苏联总工程师切断了虫洞，将所有人困死在星系中。几十年后，所有人类最终死亡，只剩下机器在运转。

### **角色**

* **主角**：总工程师的克隆体，没有本体的记忆。拥有工程天赋和“既视感”（肌肉记忆）。被设定为苏醒的“审判者”和“收尾人”。  
* **布兰妮**：无实体的 AI 助手。表层人格是总工程师女儿的天真性格，底层是冷酷的工业调度系统。两者经常发生冲突（人性 vs 程序）。  
* **敌人**：死手系统 (The Dead Hand)。美苏双方遗留的自动化防御网络，忠实地执行着几十年前的最后指令，进行着一场没有观众的战争。

具体开发内容参考文档 Biome Design & Planning

## **5\. 结局分支 (Endings)**

* **结局 A：共生**  
  * 抹除布兰妮的人性，主角与工业核心融合，成为永恒的守墓人，在机械乌托邦中孤独终老。  
* **结局 B：逃逸**  
  * 资源不足以传送实体。将布兰妮意识与警告数据发射回地球，主角留在星系坦然赴死。  
* **结局 C：归乡**  
  * **条件**：研发出“西格玛稳定剂”，掌控美苏双方顶级科技。  
  * **结果**：重启虫洞，带着清洁能源和布兰妮返回地球，修正历史的错误。

## **6\. 美术与音效指导 (Art & Audio Direction)**

### **视觉关键词**

* **UI**：示波器波形、辉光管、实体按钮、低分辨率单色 CRT。  
* **材质**：混凝土、生锈金属、霜冻、油污、电木、发黄的塑料。  
* **特效**：引力透镜效果（西格玛粒子）、磁带噪点、信号干扰、体积雾。

### **音效设计**

* **环境音**：低沉的工业轰鸣、风吹过金属废墟的啸叫、电流声。  
* **UI音效**：沉重的物理开关声、继电器吸合声、硬盘读写声。  
* **配乐**：模拟合成器 (Analog Synth)、大提琴低音、甚至苏联合唱团的采样与电子乐的混音。



## 已实现：

* 配好了基本的模组环境；
* 编写了一个 Core 模组，用于部分逻辑的具体实现；
* 编写了一个实时通信 UI 模组，用于剧情对话和展示信息；
* 有完全重写的基本世界生成逻辑，可以简略地生成我文档内提到的几种生物群系；
* 目前重点实现了气溶胶红树林群系，做出了毒雾的设定，会根据各种条件改变雾气浓度，并施加对应的效果；
* 实现了简单的地物和小结构生成。

## 接下来的规划中，已确定的要点：
* 使用机械动力代表早期科技和共通科技，沉浸工程代表苏联科技，Oritech 模组代表美式科技；
* 在早期的发展中，使用水资源作为限制因素，前往下个星球后才可以量产水资源；
* 限制早期应力来源，玩家最初只能使用手摇，后续可以通过废墟中的科技解锁风车，大量生产应力则必须要用耗水的蒸汽引擎；
* 气溶胶红树林和玄武岩管风琴都作为此阶段的后期区域，需要一定科技才能安全进入，在这些群系中可以分别找到两方的科技蓝图，并最终制造火箭前往下个星球。
